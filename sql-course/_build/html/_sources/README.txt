ALMA SQL Course
=================

**Status:** *On Development*

The course consist in an introduction
to the Databases and SQL for some
ALMA members.

Authors
--------

The source is developed by the `Computer Systems Research Group`_.

Content
--------

 * Lecture 1 - Relational Databases: "The relational model"
 * Lecture 2 - Relational Databases: Querying relational
 * Lecture 3 - Relational Algebra: Select, Project, Join
 * Lecture 4 - Relational Algebra: Set operators, renaming, notation
 * ...

.. _`Computer Systems Research Group`: http://csgr.cl
